# Project Statement

## Problem Statement
In an increasingly digital world, students and professionals often struggle with organizing daily tasks. Complex task management software can be overwhelming and resource-heavy. There is a need for a minimal, fast, and distraction-free tool that allows users to quickly log and track tasks without the overhead of a graphical user interface (GUI).

## Scope of the Project
The project focuses on building a functional CLI-based task manager.
- **In Scope**: 
  - Creating, reading, and deleting tasks.
  - persistent data storage using text files.
  - Basic input validation.
- **Out of Scope**: 
  - User authentication (login/signup).
  - Cloud synchronization.
  - Graphical User Interface (GUI).

## Target Users
- **Students**: For tracking homework and study schedules.
- **Developers**: Who prefer staying in the terminal environment.
- **Minimalists**: Users looking for lightweight software alternatives.

## High-Level Features
- **CRUD Operations**: Create, Read, and Delete functionalities.
- **Auto-Save**: Instant persistence of data changes.
- **Menu-Driven Navigation**: Intuitive command loop for interaction.
